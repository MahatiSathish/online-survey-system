# Online Survey System with Question Import

## 🚀 Features

* User authentication (JWT)
* Create, edit, delete surveys
* Import questions via CSV/XLSX
* Public survey response (no login required)
* Cloud database using Supabase PostgreSQL

## 🛠 Tech Stack

* Frontend: React + Tailwind CSS
* Backend: Node.js + Express
* Database: Supabase (PostgreSQL) + Prisma

## ▶️ Run Locally

### Backend

cd server
npm install
npm run dev

### Frontend

cd client
npm install
npm run dev

Open: http://localhost:5173

## 📌 Project Highlight

Supports dynamic survey creation with CSV/XLSX import and public response collection.
